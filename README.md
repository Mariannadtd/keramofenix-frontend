# 🚪 КерамоФеникс — e-commerce фронтенд на Vue 3 + Firebase

**Дверной Ренессанс** — полноценный e-commerce фронтенд для продаж межкомнатных и входных дверей, напольных покрытий и фурнитуры.  
Проект построен на **Vue 3**, **Pinia**, **Firebase** и **Cloudflare Workers**, включает кастомную корзину, динамический подбор размеров дверей и погонажа, админ-панель и интеграцию с Telegram.

## ⭐ Основные фичи

### 1️⃣ Умная корзина
- Добавление, удаление и изменение количества товаров  
- Живой пересчёт итоговой стоимости  
- Поддержка разных типов товаров: двери, фурнитура, напольные покрытия  
- Составные позиции (двери + погонаж)

---

### 2️⃣ Система подбора размера двери и погонажа
Пользователь выбирает размер: **60 / 70 / 80 / 90**, после чего система предлагает:

- дверную коробку  
- доборы разных ширин  
- наличник  
- плинтус  
- притворную планку  

Каждая позиция рассчитывается автоматически по формуле `qty × price`.  
Поддерживаются комментарии к погонажу и живой пересчёт итоговой стоимости.

---

### 3️⃣ Оформление заказа
- Заказ сохраняется в **Firebase Firestore**  
- Параллельно отправляется уведомление менеджерам в **Telegram Bot**  
- Все секреты вынесены в Cloudflare Worker через **Environment Variables**

---

### 4️⃣ Админ-панель
- Авторизация через **Firebase Auth**  
- Проверка прав через **custom claims**  
- Возможности:
  - добавление и редактирование товаров  
  - загрузка изображений через Cloudinary  
  - отметки hit / exhibit  
  - расширенные характеристики  
  - поиск по каталогу  
  - просмотр и удаление заказов  

---

### 5️⃣ Cloudflare Worker (безопасная интеграция)
- Принимает POST-запрос от фронта  
- Формирует читабельное сообщение для Telegram  
- Отправляет в Telegram Bot API  
- **Все секреты (BOT_TOKEN, CHAT_ID)** хранятся как *Cloudflare Secrets*  
- В репозитории лежит только `worker.example.js` без конфиденциальных данных

---

### 6️⃣ Firebase — полный стек
Используются сервисы:

- **Firestore** — каталог, заказы  
- **Storage** — изображения товаров  
- **Hosting** — прод-сайт  
- **Security Rules** — защита данных  
- **AppCheck + reCAPTCHA v3** — защита от ботов  
- **Functions** — серверная логика (опционально)
